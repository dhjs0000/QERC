# 多码识别系统

这是一个基于网页的条形码和二维码批量识别系统。用户可以通过上传图片或使用摄像头实时识别多个条形码和二维码，并将结果导出为 XML 格式。

## 主要功能

1. 图片上传识别
   - 支持上传单张或多张图片
   - 支持拖拽上传
   - 支持常见图片格式（JPG, PNG, BMP等）

2. 摄像头实时识别
   - 支持开启摄像头实时识别
   - 可同时识别多个码
   - 自动保存识别结果

3. 码类型支持
   - 一维条形码（Code128, EAN-13, UPC等）
   - 二维码（QR Code）
   - DataMatrix码

4. 结果导出
   - 导出为XML格式
   - 包含码类型、码内容、识别时间等信息
   - 支持批量导出

## 技术栈
- 前端：HTML5, CSS3, JavaScript
- 识别库：ZXing.js
- 文件处理：File API
- 摄像头调用：WebRTC
